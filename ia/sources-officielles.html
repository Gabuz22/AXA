Sources officielles

Pointeurs de navigation vers des autorités publiques. Aucun contenu réglementaire n'est stocké ici : ils indiquent où vérifier une matière évolutive. La notice PDF AXA fait foi pour le contrat.

  • Code de la sécurité sociale (Légifrance) — Protection sociale, retraite, maladie. — Code de la sécurité sociale (Légifrance)

Par concept métier

  • Invalidité (matière évolutive) : Code des assurances (Légifrance) → Ameli (Assurance Maladie) → Service-Public.fr → Légifrance
  • Décès (matière évolutive) : Code des assurances (Légifrance) → Code général des impôts (Légifrance) → BOFiP-Impôts → Service-Public.fr
  • Décès accidentel (matière évolutive) : Code des assurances (Légifrance) → Service-Public.fr
  • Accident : Code des assurances (Légifrance) → Service-Public.fr
  • Hospitalisation (matière évolutive) : Ameli (Assurance Maladie) → Code de la sécurité sociale (Légifrance) → Service-Public.fr
  • Incapacité temporaire (matière évolutive) : Ameli (Assurance Maladie) → Code de la sécurité sociale (Légifrance) → Service-Public.fr
  • Carence / délai d'attente : Code des assurances (Légifrance)
  • Rachat (matière évolutive) : Code général des impôts (Légifrance) → BOFiP-Impôts → impots.gouv.fr → ACPR → Code des assurances (Légifrance)
  • Souscription & adhésion (matière évolutive) : Code des assurances (Légifrance) → Service-Public.fr → ACPR
  • Âge (matière évolutive) : L'Assurance retraite (CNAV) → Service-Public.fr → Code de la sécurité sociale (Légifrance)
  • Suicide : Code des assurances (Légifrance) → Légifrance
  • Bénéficiaire (matière évolutive) : Code général des impôts (Légifrance) → BOFiP-Impôts → Code des assurances (Légifrance) → Service-Public.fr
  • Fiscalité (matière évolutive) : BOFiP-Impôts → impots.gouv.fr → Code général des impôts (Légifrance) → Service-Public.fr
  • Fin de garantie : Code des assurances (Légifrance)
  • Association / ANPERE : Légifrance → Service-Public.fr
